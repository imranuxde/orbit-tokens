# Orbit Design System - Typography Tokens

Design tokens for the Orbit Design System at Grameenphone.

## Contents

- `orbit-typography-tokens.json` - Typography tokens in DTCG format

## Fonts

- **Primary:** Inter
- **Bangla:** Kohinoor Bangla  
- **Monospace:** Roboto Mono

## Token Groups

- Component (XSmall → XLarge)
- Body (Small → XLarge)
- Title (Small → XLarge)
- Headline (Small → Large)
- Display (Small → Large)
